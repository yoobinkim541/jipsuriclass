# 간편 자가진단(집 단면도) 방 이미지

아래 6개 파일을 이 폴더(public/diagnosis/)에 넣으면 메인 #symptoms 섹션의
집 단면도 각 공간 배경으로 사용됩니다. (없으면 어두운 폴백 배경으로 표시)

| 파일명          | 공간            | 권장 이미지              |
|-----------------|-----------------|--------------------------|
| roof.webp       | 물·누수(지붕)   | 어두운 지붕/빗속 지붕     |
| bathroom.webp   | 욕실            | 다크 욕실(욕조·세면대)    |
| kitchen.webp    | 주방·설비       | 다크 주방(싱크·인덕션)    |
| door.webp       | 문·창문         | 다크 현관/문·창문 장면    |
| electric.webp   | 전기            | 다크 거실(조명·스위치)    |
| wall.webp       | 벽·바닥·천장    | 다크 벽/바닥             |

- 형식: webp 또는 jpg(파일명 확장자만 코드와 맞추면 됨). 가로형 권장(예: 1200×800+).
- jpg로 넣을 경우 src/App.tsx 의 HOUSE_IMG 확장자를 .jpg로 바꾸면 됩니다.
