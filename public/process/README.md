# 작업 절차(메인 #process) 단계 사진

아래 7개 파일을 이 폴더에 넣으면 작업 절차 섹션에 사용됩니다.
(없으면 임시로 기존 사진으로 폴백)

| 파일명       | 단계/용도        | 내용                          |
|--------------|------------------|-------------------------------|
| step1.webp   | 01 사진 상담     | 벽 얼룩을 휴대폰으로 촬영      |
| step2.webp   | 02 증상 확인     | 클립보드(사진)로 증상 설명    |
| step3.webp   | 03 현장 방문     | 천장 누수 가리키는 점검       |
| step4.webp   | 04 견적 안내     | 견적서 + 계산기               |
| step5.webp   | 05 시공·확인     | 작업자 벽 미장                |
| step6.webp   | 06 사후 관리     | 완공된 거실                   |
| house.webp   | 좌측 배경 일러스트 | 3D 주택(흰 배경)             |

- 형식 webp/jpg/png 무관(확장자만 코드와 맞추면 됨, 기본 .webp).
- 가로형 권장. 코드가 자동 최적화는 안 하니 너무 크면 알려주세요(webp 변환해 드림).
