# 모바일 히어로 배경 — 올리모델링 3D 목업

모바일(≤720px) 히어로의 풀블리드 배경으로 아래 5장이 4초마다 회전합니다.
세로(포트레이트) 렌더가 적합합니다.

| 파일명 | 공간 | 비고 |
|---|---|---|
| `bedroom.webp`  | 침실   | 1번 — 침대+드레스룸 |
| `bathroom.webp` | 욕실   | 2번 — 샤워부스+세면대 |
| `hallway.webp`  | 복도/현관 | 3번 — 거울벽 복도 |
| `kitchen.webp`  | 주방·다이닝 | 4번 — 아일랜드+식탁(시티뷰) |
| `living.webp`   | 거실   | 5번 — 소파+시티뷰 |

- 원본은 png/jpg 아무거나 OK → `sharp`로 폭 ~900px webp(q78)로 변환해 사용.
- 파일이 없으면 코드가 기존 시공 사진(case 이미지)으로 자동 폴백합니다(onError).
- 연결 코드: `src/App.tsx`의 `heroMobileBackdrops`.
